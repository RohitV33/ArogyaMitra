# 🧘‍♂️ ArogyaMitra – AI Driven Fitness & Wellness Platform

ArogyaMitra is an **AI-powered fitness and wellness platform** designed to help users maintain a healthy lifestyle through personalized workouts, nutrition guidance, and smart health tracking.

The platform combines **Artificial Intelligence, Fitness Analytics, and Smart Recommendations** to provide users with customized health insights.

---

## 🚀 Features

* 🏋️ **Personalized Workout Plans**
* 🥗 **AI-Based Nutrition Recommendations**
* 📊 **Health & Fitness Analytics Dashboard**
* ⏰ **Smart Health Reminders**
* 🤖 **AI Wellness Assistant**
* 📅 **Workout Scheduling**
* 📈 **Progress Tracking**

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/RohitV33/ArogyaMitra.git
cd ArogyaMitra
```

### 2️⃣ Setup Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3️⃣ Run Backend Server

```bash
uvicorn main:app --reload
```

Server will start at:

```
http://127.0.0.1:8000
```

API Docs:

```
http://127.0.0.1:8000/docs
```

---

### 4️⃣ Run Frontend

Open the frontend folder and run the frontend server.

```
frontend/index.html
```
